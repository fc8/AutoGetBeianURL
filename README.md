# 批量获取域名备案信息
## 介绍
在资产收集时，省去需要一个一个去点详情复制网址的操作。
## 使用方式
1. 导入油猴脚本中
2. 手动搜索资产
3. 在搜索框下会创建一个文本框展示搜索到的网址
4. 直接复制即可，下一页会自动替换文本框内容
